# Día 12: Módulos 📦

Hoy he dado un salto de gigante al aprender sobre **Módulos**. Un módulo es un archivo que contiene un conjunto de funciones que puedes incluir en tu aplicación. Esto permite dividir el código en partes manejables e incorporar herramientas potentísimas creadas por la comunidad de Python.

### 🧠 Conceptos Aprendidos
- **Creación de Módulos:** Cómo separar mi propio código en distintos archivos `.py`.
- **Importación (`import`):** - `import modulo` (importa todo).
  - `from modulo import funcion` (importa algo específico, ahorra memoria).
  - `import modulo as md` (crea un alias para escribir menos).
- **Módulos Integrados (Built-in) Clave:**
  - `math`: Operaciones matemáticas avanzadas, trigonometría, estadística.
  - `string`: Colecciones de caracteres (letras, números, puntuación).
  - `random`: Generación de números aleatorios y selección al azar (¡muy divertido de usar!).
  - `os` y `sys`: Interacción con el sistema operativo y el entorno.

### 🛠️ Aplicación en IA
La Inteligencia Artificial en Python **existe gracias a los módulos**. 
Nadie programa una red neuronal desde cero usando matemáticas básicas. Lo que hacemos es poner `import tensorflow` o `import torch` al principio de nuestro archivo, y de repente tenemos acceso a miles de horas de trabajo de los mejores ingenieros de Google y Meta. 
Librerías como **NumPy** (para matemáticas con matrices) o **Pandas** (para analizar datos) son simplemente módulos gigantes que importamos a nuestros proyectos.
