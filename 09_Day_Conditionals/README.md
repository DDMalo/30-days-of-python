# Día 9: Condicionales (If/Else) 🚦

Hoy he aprendido a darle lógica y capacidad de decisión a mis programas usando **Condicionales**. Sin ellos, el código se ejecutaría siempre igual; con ellos, el programa puede reaccionar de forma distinta según los datos de entrada.

### 🧠 Conceptos Aprendidos
- **Estructura Básica:**
  - `if`: "Si pasa esto..." (ejecuta el bloque).
  - `elif`: "Si no, pero pasa esto otro..." (condición alternativa).
  - `else`: "Si no ha pasado nada de lo anterior..." (caso por defecto).
- **Indentación:** En Python es obligatoria. El código dentro del `if` debe estar sangrado (tabulado) para que funcione.
- **Operadores Lógicos:** `and`, `or`, `not` para combinar múltiples condiciones.
- **Operadores de Comparación:** `==` (igual), `!=` (diferente), `>`, `<`, `>=`, `<=`.
- **Condicionales Anidados:** Poner un `if` dentro de otro `if`.

### 🛠️ Aplicación en IA
Los condicionales son la forma más primitiva de "Inteligencia".
1.  **Sistemas Basados en Reglas:** Los primeros chatbots eran solo miles de `if` y `else` anidados.
2.  **Funciones de Activación:** En las neuronas artificiales, funciones como **ReLU** son puramente condicionales: `if x > 0: return x; else: return 0`.
3.  **Filtrado de Datos:** "Si la imagen es demasiado oscura (`if brightness < 0.2`), descartarla del set de entrenamiento".
