# 🐍 30 Days of Python Challenge

¡Hola! 👋 Soy **David Donoso Malo**, estudiante de **Ingeniería Informática**. Este repositorio es mi bitácora personal para el reto de 30 días de Python.

Mi objetivo es dominar las bases de Python para dar el salto a mi verdadera pasión: la **Inteligencia Artificial** 🤖.

---

## 🎯 Objetivos del Reto
- [x] Dominar la sintaxis básica (Variables, Operadores).
- [x] Controlar las Estructuras de Datos fundamentales (Listas, Tuplas, Diccionarios, Sets).
- [ ] Implementar lógica de control de flujo avanzada (Bucles, Funciones).
- [ ] Mantener una racha de commits diaria para demostrar constancia.
- [ ] Preparar el terreno para librerías de IA (NumPy, Pandas, PyTorch).

---

## 📂 Estructura del Proyecto
Cada carpeta contiene los ejercicios, notas teóricas y soluciones correspondientes al día del reto. El código está comentado paso a paso.

- **Días 1-3:** Fundamentos y Operadores.
- **Día 4:** Procesamiento de texto (Strings).
- **Días 5-8:** Estructuras de Datos (Listas, Tuplas, Sets, Diccionarios).
- **Día 9:** Lógica y Condicionales.
